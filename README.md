# Vision-Language-and-Intelligent-Learning (VLIL)
 THE VISION, LANGUAGE AND INTELLIGENT LEARNING (VLIL) THRUST AREA GROUP , AT AMRITA UNIVERSITY  PURSUES FUNDAMENTAL RESEARCH AT THE INTERSECTION OF COMPUTER VISION, NATURAL LANGUAGE PROCESSING AND MACHINE LEARNING. WE AIM TO CREATE INTELLIGENT SYSTEMS THAT CAN LEARN FROM VAST AMOUNTS OF VISUAL , TEXTUAL AND BIOMEDICAL SIGNAL  INFORMATION, THAT CAN INTEGRATE AND ENHANCE HUMAN EXPERIENCES, AND THAT CAN RESOLVE COMPLEX TASKS THAT TYPICALLY REQUIRE HUMAN INTELLIGENCE.
